# dualsense-pc-node

Proyecto para controlar el dualsense de ps5 en pc mediante node.js.

## Funciones

Cambiar color del touchpad led mediante un selector rgb.

Conexion USB.

Conexion bluetooh proximamente.

## Dependecias

node-hid

electron
